# 康治本伤寒论
## 泻心汤
### 甘草泻心汤
甘草四两炙，黄连三两，黄芩三两，干姜三两，大枣十二枚擘，半夏半升洗
### 半夏泻心汤
半夏半升洗，黄连三两，黄芩三两，人参三两，干姜三两，甘草三两炙，大枣十二枚擘
### 生姜泻心汤
生姜四两切，黄连三两，黄芩三两，人参三两，甘草三两炙，大枣十二枚擘，半夏半升洗
