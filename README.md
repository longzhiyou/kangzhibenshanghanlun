# 康治本伤寒论


## 概念
### 变证
> 异常的症状，主要是汗、吐、下后，病未愈出现的症状。

- 条文对应的大多是以【全部生药命名】或【多个生药+修饰语】的方剂

### 正证
> 正常症状，也可以叫【主证】

- 条文对应的大多是以【非全部生药命名】或【单个生药+修饰语】的方剂

# 【甘草】
- 汤形成的第一原则
- 国老
- 上历观此诸方。无论急迫，其他曰痛、曰厥、曰烦、曰悸、曰咳、曰上逆、曰惊狂、曰悲伤、曰痞硬、曰利下，皆甘草所主。而有所急迫者也，仲景用甘草也；其急迫剧者，则用甘草亦多。不剧者，则用甘草亦少。由是观之，**甘草之治急迫也明矣**。古语曰∶病者苦急，急食甘以缓之。其斯甘草之谓乎？仲景用甘草之方甚多，然其所用者，不巡前证，故不枚举焉。


## 【芍药、甘草】
- 缓和脚挛急（下肢肌肉抽筋）状态
- 今日平滑肌痉挛引起的腰痛可以使用此方
- 此方包含于各种汤中的基本方
- 此方具有改善骨骼【肌肉痉挛与腹痛】的功能

# 桂枝
- 主治冲逆也，旁治奔豚头痛、发热恶风、汗出身痛。
## 【桂枝、甘草】
- 康治本**凡使用桂枝必用甘草**，但没出现此书。
- 发汗过多，其人叉手自冒心、心下悸欲得按者，桂枝甘草汤主之。
- 与发汗及其相关、有改善【心悸亢进】的作用
# 茯苓
- 主治悸及肉 、筋惕也。旁治小便不利、头眩烦躁
### 【茯苓、桂枝、甘草】
- 推断茯苓：具有改善身体强烈动悸、镇定不安、兴奋的作用
#### 【茯苓、桂枝、甘草、大枣】
- 20.发汗后，脐下悸，欲作奔豚者，茯苓桂枝甘草大枣汤主之
- 动悸在下腹部比较明显，像有什么东西从下腹部往上冲撞的感觉而产生痛苦

## 【茯苓、白术】
- 21.发汗，若下之后，**心下逆满**，气上冲胸，**起则头眩**者，茯苓桂枝甘草白术汤主之。
- 太阳病发汗，汗出不解，其人仍发热，**心下悸、头眩**、身目闰动、振振欲僻地者，真武汤主之”
#### 【茯苓、桂枝、甘草、白术】
- 21.发汗，若下之后，心下逆满，气上冲胸，起则头眩者，茯苓桂枝甘草白术汤主之。 
- 好像有什么东西堵塞于心下（心窝）有冲撞胸部的感觉，而产生痛苦，以至于头部也受到了影响而产生头眩（立起则眩晕）

## 重要信息
### 鞭避讳坚，杨坚，隋朝皇帝
### 表里同时发病，叫做`合病`
### 如果先有表，后有里，表证未罢，里病又生，叫做`并病`。
### 去桂的时机：
> 服桂枝汤，或下之后，仍头项强痛，翕翕发热，无汗，`心下满微痛，小便不利者`，桂枝去桂加白术茯苓汤主之
